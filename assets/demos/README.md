# Demo Assets

Drop short product clips/GIFs here to power the homepage demo gallery.

Expected filenames:
- `create-task.gif`
- `live-apply.gif`
- `task-review.gif`

Tips:
- Keep files under ~8 MB each for fast page loads.
- 16:9 or 4:3 aspect ratios work best.
- Prefer recordings that show cursor movement and one full action flow.
