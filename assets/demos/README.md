# Demo Assets

Drop short product clips/GIFs here to support the secondary workflow-clips section on the homepage.

Expected filenames:
- `create-task.gif`
- `live-apply.gif`
- `task-review.gif`

Recommended use:
- Screenshots should carry the primary proof.
- GIFs should add motion, not explain the whole product.
- If a clip feels noisy or hard to read, leave the placeholder instead.

Tips:
- Keep files under ~8 MB each for fast page loads.
- 16:9 or 4:3 aspect ratios work best.
- Prefer recordings that show one clean action flow.
